# 硬盘

## 硬盘的分类

硬盘分为固态硬盘（SSD 盘）、机械硬盘（HDD）、混合硬盘（HHD）三种。

SSD（Solid State Disk）采用闪存颗粒来存储，

HDD（Hard Disk Driver）采用磁性碟片来存储，

HHD（Hybrid Hard Disk)是把磁性硬盘和闪存集成到一起的一种硬盘。绝大多数硬盘都是固定硬盘，被永久性地密封固定在硬盘驱动器中。

## 硬盘接口

硬盘接口通常分为五种类型：

* IDE接口硬盘

Integrated Drive Electronics, 本意是指把控制器与盘体集成在一起的硬盘驱动器，目前多用于家用产品。又名ATA（Advanced Technology Attachment）。

* SATA接口硬盘

Serial ATA，串口硬盘. SATA以它串行的数据发送方式得名，是市场上最普及的接口类型。

* SCSI接口硬盘

IDE口太慢，出线了高速的SCSI。SCSI并不是专为硬盘设计的，实际上它是一种总线型接口。独立于系统总线工作。主要应用于服务器市场

* SAS接口硬盘。

串行连接SCSI，SAS的接口技术可以向下兼容SATA。

* 光纤通道硬盘

FC（Fibre Channel）高端服务器，价格昂贵

## X86

X86是一个微处理器执行的计算机语言指令集，是兼容8086，286，386等的一套通用的计算机指令集合。

x86-64，是**64位**微处理器架构及其相应指令集的一种。也是Intel x86架构的**延伸**产品。“x86-64”1999由AMD设计，AMD 首次公开 64 位集以扩充给 IA-32，称为 x86-64（后来改名为 AMD64）。 其后也为Intel所采用，现时英特尔称之为“Intel 64”.

AMD64和Intel64基本上一致，因此“x86-64”被不时地使用，缩写为“x64”

目前的PC架构绝大多数都是Intel的X86架构

# Linux系统

## 基本命令

终端，这里输入的命令就是我们shell的命令，一般$表示的是普通用户，而#表示的是超级用户（root）

以debian为例

```shell
# 检查软件包
$ cat /etc/apt/sources.list
# 更新软件包
$ apt update && apt update -y && apt autoremove -y

# wget下载软件包
# -q：不显示指令执行过程；
# -O：下载并以指定的文件名保存；
$ wget -qO- https://get.casaos.io | bash

```

## shell脚本

```bash
#!/bin/bash
#获取字符串长度
str="123456"
echo "${#str}"
```

\#!是一个约定的标记，告诉系统这个脚本需要什么解释器来执行，后面路径指明了解释器的具体位置

```sh
# 执行脚本 需要改变文件的权限
$ chmod 777 test.sh
$ /bin/bash/test.sh
```
